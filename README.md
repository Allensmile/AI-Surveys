# AI-Surveys

本repo主要整理AI相关领域的一些综述，起因是看到了 [ml-survey](https://github.com/eugeneyan/ml-surveys) 这个非常棒的项目。

目前添加了『自然语言处理』模块的部分觉得不错的综述。

欢迎有兴趣的小伙伴们一起整理。
